# 🎮 我的世界 Python 版本

一个基于 Python 的 3D 体素游戏引擎，完全模仿 Minecraft 的画面风格和游戏机制。

## ✨ 主要功能

- ✅ **3D 体素渲染** - 使用 OpenGL 绘制高保真像素立方体
- ✅ **程序化地形生成** - 类似 Minecraft 的随机世界生成
- ✅ **多种方块类型** - 泥土、草、石头、沙子、木头等
- ✅ **自由摄像机控制** - WASD 移动，右键鼠标旋转视角
- ✅ **实时渲染** - 60 FPS 流畅画面
- ✅ **高分辨率支持** - 1920x1080 高清画面

## 🎮 游戏操作

| 按键 | 功能 |
|------|------|
| **W** | 向前移动 |
| **A** | 向左移动 |
| **S** | 向后移动 |
| **D** | 向右移动 |
| **Space** | 向上移动 |
| **Shift** | 向下移动 |
| **右键 + 鼠标** | 旋转视角 |
| **ESC** | 退出游戏 |

## 📦 环境需求

- Python 3.8+
- pip 包管理器

## 🚀 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/dssxsr/minecraft-python.git
cd minecraft-python
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 运行游戏

```bash
python main.py
```

## 🧠 核心类说明

### Block 类
- 代表游戏中的单个方块
- 处理方块的顶点、边和面
- 负责方块的 3D 渲染

### World 类
- 管理整个游戏世界
- 使用噪声算法生成随机地形
- 决定每个位置放置什么方块类型

### Camera 类
- 控制玩家的视角和移动
- 处理键盘输入和鼠标旋转
- 管理 OpenGL 摄像机矩阵
